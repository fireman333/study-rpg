**Root cause**:

**Fix approach** (in `/tmp/medexam2-fix-3-truncated.py`):
1. 定位 stem marker + next-Q marker（允許跨頁）
2. union 兩個 marker 之間的所有 embedded image bbox
3. 用 `page.get_pixmap(clip=bbox, matrix=300dpi)` render page **slice**（不是抽 embedded image xref）
4. 跨頁時 vstack 兩個 slice

關鍵差異：page slice render 對 embedded image tile / 多 xref 自動 immune，因為直接從 PDF raster page 取。

**Lesson 給未來補圖任務**:
